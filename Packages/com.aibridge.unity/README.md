# Unity AI Bridge — UPM Package

This is the Unity Package Manager (UPM) package for Unity AI Bridge.

For full documentation, setup guides, and IDE configuration, see the [main README](https://github.com/butterlatte-zhang/unity-ai-bridge).

## Installation

**Via Package Manager:**
1. Open Unity → Window → Package Manager
2. Click "+" → "Add package from git URL..."
3. Enter: `https://github.com/butterlatte-zhang/unity-ai-bridge.git?path=Packages/com.aibridge.unity`

## Requirements

- Unity 2022.3 or later
- Any render pipeline (URP, HDRP, Built-in)

> **Note:** This is an editor-only package. All assemblies target the Unity Editor platform and are stripped from player builds automatically.
