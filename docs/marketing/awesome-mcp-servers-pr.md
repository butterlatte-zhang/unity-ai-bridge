## PR Title

Add Unity AI Bridge — 65-tool MCP server for Unity Editor

## PR Body

Adds [Unity AI Bridge](https://github.com/butterlatte-zhang/unity-ai-bridge) to the list.

**What it does:** Gives AI agents full access to the Unity Editor via MCP — 65 tools covering scene management, GameObject/component CRUD, asset operations, prefab editing, profiling (snapshot, hotpath, GC), light probes, screenshot, runtime query/invoke, script execution, reflection, and more.

**Key features:**
- Unity 2022.3 LTS+ (not limited to Unity 6)
- File-based IPC — zero dependencies, no open ports
- Works with Claude Code, Cursor, Copilot, Windsurf, Claude Desktop
- Apache 2.0 licensed
